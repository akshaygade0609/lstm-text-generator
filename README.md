lstm-textLSTM Text gGenerator
This project implements a text generation model using LSTM (Long Short-Term Memory).

The model is trained on a subset of Shakespeare's works and generates new text based on a given seed input.

The implementation is built using PyTorch.

Features
Text preprocessing
Character-level tokenization
LSTM-based neural network
Model training using PyTorch
Text generation from seed input
Dataset
Shakespeare Dataset from Project Gutenberg:

https://www.gutenberg.org/files/100/100-0.txt

Note:
Due to local hardware memory limitations, a smaller subset of the dataset was used for training.

Requirements
Install dependencies using:

pip install -r requirements.txt
